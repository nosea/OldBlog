在win7上搭建Android开发环境

Dec 10th, 2012 10:13 pm

一.System Requirements, 来自官方文档
Eclipse IDE Eclipse 3.6.2 (Helios) or greater Note: Eclipse 3.5 (Galileo) is no longer supported with the latest version of ADT. Eclipse JDT plugin (included in most Eclipse IDE packages) JDK 6 (JRE alone is not sufficient) Android Development Tools plugin (recommended) *Not compatible with GNU Compiler for Java (gcj)

二.安装Java SDK
1.下载Java SDK, 找到最近更新JDK 6即可.
2.安装目录自选，建议路径不留空格, 留意安装程序是否设置了环境变量等, 没有则自己设置.

三.安装Eclipse
1.下载Eclipse Helios, 下载Eclipse IDE for Java Developers.
2.安装目录自选, 解压到一个目录就行了. 简单起见, 建议根目录.

四.安装Android SDK
1.下载Android SDK.
2.有两种方法可以安装: ADT Bundle 和 SDK Tools Only. 还是直接下SDK Tools Only吧.
3.下载后解压到任意目录, 建议根目录.
4.进入android-sdk-windows, 运行SDk Setup.exe, 全接受, 安装.
5.把android-sdk-windows下tool目录加入环境变量.

五.配置Eclipse
1.安装ADT插件, 参考方法见官方文档
2.按照自己需求, 增加所需的虚拟机即可.
